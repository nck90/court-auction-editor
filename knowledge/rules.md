# 법원 경매 공고 편집 기준 (2023.12.04 버전)

이 문서는 InDesign 편집자가 실제로 따르는 규칙을 구조화한 것이다. 자동화 파이프라인은
이 규칙을 최대한 충실히 재현해야 한다.

---

## 1. 표 구조

| 칸 | 내용 |
|----|------|
| 사건번호 | `YYYY타경NNNNN`, 필요 시 `[중복]`, `[병합]` 표시 추가 |
| 물건번호 | 숫자 |
| 소재지 및 면적[㎡] | 주소 + 면적 (동·호 띄어쓰기 유지, 층·㎡는 붙여쓰기) |
| 용도 | 아파트 / 연립주택,다세대등 / 상가,오피스텔등 / 근린시설 / 단독주택 / 대·임야·전·답·잡종지·공장용지·도로·묘지 / 공장 / 자동차 / 선박 / 기타 |
| 감정평가액 / 최저매각가격 [단위:원] | 상/하 2줄 |
| 비고 | 일괄매각, 지분매각, 제시외건물포함, 농지취득자격증명요 등 |

## 2. 그룹 분류 (인쇄 순서)

1. `[아파트]`
2. `[연립주택/다세대/빌라]`
3. `[단독주택,다가구주택]`
4. `[상가/오피스텔,근린시설]`
5. `[대지/임야/전답]`
6. `[기타]`

자동차/선박/건설기계/항공기는 별도 공고, 부동산 공고에서는 제외.

그룹 내부 정렬: 사건번호 오름차순 → 물건번호 오름차순.

## 3. 소재지 표기 규칙

### 3-1. 주소 접두어 제거
- `경기도/강원특별자치도/…/서울특별시/부산광역시/…/제주도` 등 **-도/-특별자치도/-광역시/-특별시 접두어 제거**.
- **예외**: `세종특별자치시`는 유지.

### 3-2. 동소(주소 동일) 처리
- 같은 물건(동일 entry) 내 연속되는 소재지의 공통 접두가 **리/동/로/길/가** 또는 번길(예: `와수리 1206-7` 다음 `와수리 1206-43`) 수준까지 일치하면 뒤 항목은 `동소 <마지막 고유 번지>`로 치환.
- 행정구역(면/읍/구) 수준까지만 같고 리/동이 다를 경우에는 **그대로 원주소 유지**. (예: `양주시 남면 한산리 578` → `양주시 남면 신산리 320` 이면 두 번째는 `동소` 불가.)

### 3-3. 띄어쓰기
- 주소의 **동·호** 는 띄어쓰기 유지 (예: `5동 5층 501호`).
- **층+면적(㎡) 은 붙여쓰기** (예: `1층102㎡`).
- 쉼표 뒤 공백 제거, 괄호 주변 공백 제거 → 대괄호로 변환 `( ) → [ ]`.

### 3-4. 조표제(91-918호 등)
- 조표제가 있으면 소재지에 삽입.

### 3-5. 주식회사 축약
- `주식회사 ○○` → `[주]○○`.

### 3-6. 지분
- `갑구7번 정선영 지분 2분의 1 전부` → `갑구7번정선영1/2지분전부`.
- `지분 2분의 1` → `1/2지분`.
- 지분 정보는 소재지에만 기록하고 비고에는 중복 기재 안 함.

### 3-7. 토지별도등기있음
- 소재지 또는 비고에 기입 가능.

## 4. 건물 상세 (구조+면적)

### 4-1. 건축자재/구조 표현 제거
- 비고·특이사항이 없으면 건축자재 전체 삭제. 삭제 대상 키워드:
  - `철근콘크리트구조/조/벽식조`, `일반철골구조`, `경량철골구조`, `철골조`,
  - `시멘트블록조`, `시멘트벽돌조`, `벽돌조`, `블록`, `블록조`, `샌드위치판넬지붕`, `샌드위치 판넬`,
  - `슬래브지붕/슬라브지붕`, `판넬지붕`, `판넬` 등.

### 4-2. 용도 중복 제거
- 소재지의 용도명이 여러 번 등장하면 첫 한 번만 유지하고, 사건번호 용도와 같으면 모두 제거.
- 일반 건축물의 경우 관찰평가·건축자재는 줄 수 조정을 위해 전체 삭제.

### 4-3. 층별 면적 포맷
- `N층 [용도]M㎡` 또는 `N층M㎡` 형태 (층과 ㎡는 붙여 씀).
- `단층공장` 같이 층수 없음 건물은 `단층` 접두 유지.
- 같은 면적이 연속된 층이면 `2∼4층각463.81㎡`처럼 묶음.
  - 연속된 숫자는 `-` 가 아닌 `∼` 로 잇는다 (PDF 예시: `2∼4층`).
  - 동일 면적의 층은 `,` 로 연결 `층각 M㎡` (예: `1,3층각24㎡`).
- 지하층: `지하1층214.2㎡` (숫자·동·호 띄어 안 씀).

### 4-4. 부속건물
- `부속건물 단층재배사 176㎡` 처럼 `부속건물` 접두 유지.

### 4-5. 제시외
- 소재지에 '제시외'가 있으면 **상세내역의 제시외 건물/수목 면적을 모두 더해서 대표 라벨과 함께** 기록.
  - 예: `제시외 건물등1,147.4㎡`, `제시외 계단실등254.4㎡`, `제시외 다용도실등31㎡`, `제시외 창고등11.4㎡`.
  - 라벨은 **글자수가 가장 적은 대표 건물명** 또는 가장 먼저 언급되는 주요 용도.
- `-식/-주` 단위 수목은 `제시외수목 단풍나무등 n식/주`.
- 기계기구는 상세 `n식`들을 모두 더해 `㎡` 로 적는다 (단, 숫자 없으면 이름만).

## 5. 용도 라벨 표준화

| 법원 원고 | 표기 |
|----------|------|
| 제1종근린생활시설, 제2종근린생활시설, 근린생활시설 | 근린시설 |
| 주거용 오피스텔, 업무용오피스텔 | 오피스텔 |
| 연립주택, 다세대주택, 빌라 → 한 그룹으로 묶인 경우 | `연립주택,\n다세대등` |
| 상가, 오피스텔, 근린시설이 같은 entry | `상가,오피\n스텔등` (2줄) – 단 한 줄로 충분하면 `상가,오피스텔등` |
| 단독주택 + 근린시설 | `단독주택,\n근린시설` |
| 단독주택 / 다가구주택 | `주택` |
| 기계기구목록 | `기계기구\n목록` |
| 잡종지, 공장용지, 창고용지, 체육용지, 주차장, 주유소용지, 대, 임야, 전, 답, 묘지, 도로 | 토지 코드 그대로 |
| 상세에 `공장`·`제조업소` 있음 | `공장` |
| 용도 기재 없음 + 상세에 `공장용지` | `공장용지` |
| 용도 애매 | `기타` |

용도 내용이 소재지에 포함되어 있으면 소재지에서 해당 단어를 삭제 (중복 회피).

## 6. 비고 규칙

### 6-1. 문장 끝처리
- 마침표 `.` 로 문장 구분. **마지막 문장 및 날짜의 마지막 `.` 삭제**.
- 공백 전부 제거(띄어쓰기 불가).

### 6-2. 주요 키워드 및 표기 순서
아래 순서대로 넣는다:

1. `[중복] / [병합]` 사건번호 (비고 맨 앞; 소재지 줄 수가 5 이상이면 사건번호 칸으로).
2. 일괄매각 / 지분매각 **(맨 앞)**.
3. 목록 정보: `목록3지분매각`, `목록4분묘소재` 등.
4. 제시외건물포함 **(농지취득자격증명요 보다 앞)**.
5. 농지취득자격증명요 → `농지취득자격증명요`.
6. 공유자우선매수 관련:
   - `공유자우선매수권행사에관한특별매각조건있음` (원문 그대로 있으면)
   - 아니면 `공유자우선매수신고1회제한`.
7. 지상수목포함매각.
8. 분묘1기소재.
9. 기계기구목록.
10. 기타 특이 조항 (예: 공장및광업재단저당법 제6조, 유치권신고).

### 6-3. 축약
- `농지취득자격증명원제출요(미제출시-)` → `농지취득자격증명요`.
- `농지법 제8조 제1항 제1호에 따라 농지취득자격증명원 필요하지 않음` → `농지취득증명원불요`.
- `현황추심임` → `현황축심`.
- `조경용으로조성된연못` → `조경용조성연못`.
- `더덕및나물` → `더덕등`.

### 6-4. 중복 / 불필요 삭제
- 같은 의미의 서술은 한 번만.
- `제시외건물철거명령받았으나, 현재 철거하지못한 상태는 채무자겸소유자 진술있음` →
  `현재제시외건물철거하지못한상태라는채무자겸소유자진술있음`.

### 6-5. 지분매각 조건
- `지분매각`이 비고에 있으면 `농지취득자격증명요`는 드롭하고 지분매각이 첫 자리로 간다.

### 6-6. 중복 dot 제거
- `..` 은 절대 나오면 안 됨.
- 같은 키워드 두 번 `지분매각.지분매각` 금지.

## 7. 사건번호

### 7-1. (중복), (병합) 처리
- `(중복)/(병합)` 은 `[중복]/[병합]` 로 변환.
- 소재지 줄 수가 5 미만일 때는 비고 칸 맨 앞에 `2024타경NNNN [중복]` 형태로.
- 소재지 줄 수가 5 이상이면 사건번호 칸으로 옮김.

### 7-2. 정렬
- 용도 그룹 내 사건번호 오름차순 → 물건번호 오름차순.

## 8. 자동차 / 선박 (참고, 이 파이프라인은 부동산만 처리)

- 자동차: `사용본거지`, `등록번호`, `차명`, `연식` 필수.
- 선박: `보관:○○○` 포맷.
- **부동산 공고 테이블에는 자동차/선박 entry 포함 안 함**.

## 9. 하단 문구

- 하단 문구는 고정 템플릿 (특별매각조건, 매각기일, 매각결정기일, 매각장소, 게재일자, 법원 및 지원, 사법보좌관 이름).
- 날짜 끝에는 `.` 없음 (예: `2025. 3. 20`).

## 10. Edge case 및 실제 관찰된 예시

### 10-1. 잡종지 단어 보존
- `양주시 남면 한산리 578 908㎡` + 상세 `잡종지 908㎡` → 렌더링: `양주시 남면 한산리 578 잡종지 908㎡`.
- 즉, 토지 용도 코드 (잡종지, 대, 전, 답, 임야 등)는 **주소+용도+면적** 형태로 나열.

### 10-2. 근린시설(용도 기호 `각사무소`)
- 상세에 `1층 제2종근린생활시설(사무소)116.2㎡ 2층 제2종근린생활시설(사무소)126.73㎡` 이 오면
  - 용도가 사건번호 수준에서 `근린시설` 이라면, 소재지에는 `1층116.2㎡ 2층126.73㎡ 각사무소` 로 표기.
  - 즉 용도 중복 삭제 + `각사무소`는 뒤에 붙여서 보존.

### 10-3. 다용도 층별 (유흥주점/계단실/…) 보존
- `지하1층 : 214.2㎡(유흥주점)` → `지하1층유흥주점214.2㎡`.
- 같은 층에 여러 용도가 분할 기재된 경우 용도명 괄호 제거 후 `N층<용도><면적>㎡` 형식으로 나열.
- 연속된 같은 면적층은 `2∼4층여관각463.81㎡`.

### 10-4. 창수면 포천로 2575 각 동
- `비동호 단층공장325.8㎡` → `동소 2575 비동호 단층325.8㎡` (용도는 행별 `공장`).
- 같은 주소·지번에서 동호만 바뀔 때는 지번 유지 (`2575`) 필요.
- 같은 번지 내 '다동·비동·씨동' 처럼 구분자가 있는 경우 동 이름 유지.

### 10-5. 일괄매각 + 제시외건물포함 + 기호제외
- `일괄매각. 제시외 건물 포함(기호ㄴ제외)` → `일괄매각.제시외건물포함[기호ㄴ제외]`.
- 원문에 구체 기호 제외 표기가 있으면 모두 대괄호로 이동.

### 10-6. 한 줄 용도
- `상가,오피\n스텔등` 은 원래 셀 폭이 좁아 2줄이지만, 폭이 충분하면 `상가,오피스텔등` 한 줄도 허용.
- 같은 entry 내 `상가,오피스텔,근린시설` → 표기 `상가,오피스텔등` 또는 `상가,오피\n스텔등`.

### 10-7. 공장
- 상세에 `공장` 이라는 단어가 직접 나오면 용도를 `공장`으로.

### 10-8. 농기계수리점
- 용도 매핑 없음 → 용도 `기타`, 상세에는 `단층 농기계수리점195㎡` 형태로 남김.
