검수 화면 도움말

그룹 분류 변경

  • 카드를 다른 그룹 박스로 드래그앤드롭
  • 또는 카드 우측 드롭다운으로 선택
  • 체크박스로 여러 카드 선택 → 하단 바에서 일괄 이동

내용 편집

  • 카드 클릭 또는 카드의 [편집] 버튼 → 화면 이동 없이 같은 자리에서 바로 수정
  • 편집 중에는 카드 영역을 넓혀 사건번호·금액·소재지·비고 입력칸을 크게 표시
  • 소재지는 [＋ 소재지 행 추가] 버튼으로 추가, 각 행 우측 [×]로 삭제

PDF 자동 반영

  • 모든 변경은 상단 [저장 후 PDF 재생성]을 눌러야 PDF에 반영됩니다
  • 저장 후 우측 PDF가 자동 갱신됩니다

키보드 단축키

  • / — 검색창 포커스
  • ? — 도움말 열기
  • Esc — 인라인 편집기/모달 닫기
  • Ctrl+S — 전체 저장 후 PDF 재생성
  • Ctrl+Z — 마지막 변경 취소
  • Ctrl+A (검색 결과 안에서) — 전체 선택
  • F — PDF 풀스크린 모드 토글
  • 카드 우클릭 — 빠른 작업 메뉴 (그룹 이동·삭제·편집)

특별 표시

  • 주황색 좌측 띠 — AI가 분류 어려움으로 표시한 사건 (지분매각·분묘소재·토지별도등기·제시외 등 키워드 포함). 우선 검수 권장.
  • 노란 배경 — 그룹이 변경된 사건
  • 파란 배경 — 필드(소재지·비고 등)가 편집된 사건
